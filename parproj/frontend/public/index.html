Methodology + Objectives

Concise scope summary derived from the project brief: sequential baselines, OpenMP feature extraction, CUDA preprocessing, and performance evaluation on real graphs.

Objectives

  • Implement sequential baselines for degree, betweenness, PageRank, and clustering.
  • Parallelize feature extraction using OpenMP with multi-level strategies.
  • Develop CUDA kernels for preprocessing and matrix operations.
  • Train a parallel Random Forest with tree and node-level parallelism.
  • Measure speedup, scalability, and accuracy on real-world datasets.

Methodology

  • Build CSR graph representation and adaptive CPU/GPU splitting by degree.
  • Apply OpenMP scheduling for irregular workloads and load balance.
  • Use CUDA for regular preprocessing (z-score/min-max) and GEMM-based distance.
  • Integrate CPU and GPU stages with efficient data movement.
  • Evaluate on SNAP datasets and visualize performance trends.
